## 使用步骤

1. 克隆代码:

```bash
```

2. 打开`微信Web开放者工具`（注意：须`v0.10.101100`及以上版本）

3. 添加项目

    * AppID：选`无AppID`
    * 项目名称：任意填写
    * 项目目录：path/to/your/workspace
    * 点击 `添加项目`
